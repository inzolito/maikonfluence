Overview

Phase 1 - Develop a tablet-based Open Pit Task Management solution for Ancillary Equipment

Scope

Within scope

Phase 1

Task management App for OP Ancillary equipment 

Task management Admin and User Web UI

Out of scope

Production Fleet Task Management

Other fleet and use cases (i.e. generic task management)

Background and Strategic Fit

Brazilian SmartMine 3 Migrations

  • Aligned to ‘Brazil FMS Strategy’ to migrate past Devex customers from legacy SM3 products to current MineOperate OP products
  • SM3 Extreme product was developed for OP Ancillary operations (Vale Itabira 90+ units)
  • SM3 Extreme utilising legacy EoL software & hardware
  • Key requirement is need for OP Task Management solution for Ancillary fleet


Key features

  • Create, Assign, Schedule, Complete, Maintain Tasks for Production & Ancillary Equipment
    • End User (Dispatcher, Supervisor)
    • Operator
  • Real-time data push to field devices and updates
  • Create task from mine map interface and location association
  • Equipment location tracking & job status reporting
  • Real-time KPI analytics and reporting

Deliverables

Task Management MVP

Task Management App (feature complete)

  • Time + Task App

Task Management Module

  • Administration Web UI
  • User Web UI
  • Map Features

Assumptions

  • Existing MineEnterprise Task Management (Compliance) module administration module will be leveraged.  Currently a Silverlight UI. New Task Admin Web UI to be built over this.
