Product: MineOperate Foundation : Get Started (LOADER)

Created by Usuario desconocido (dask), last modified on feb 12, 2020


Flic:

  • Loader allows operators to log Time Allocations by means of a clicker.
    • Specifically Flic (https://flic.io/shop/flic-1pack) 
      • Unsure if the new Flic-2 buttons are compatible
    • Loader interacts with the button by means of the Flic SDK (https://partners.flic.io/partners/developers)
      • Currently Loader uses cordova-plugin-flic (v 0.5.0) to interface with the SDK
    • Prior to installing Loader, the Flic App needs to be installed on the device (https://play.google.com/store/apps/details?id=io.flic.app)
      • The app will still run without the button, the user can tap on the screen instead of the button.
      • But the flic app needs to be installed to get access to the sdk.



Supported DataBase:

  • BHP Leinster Underground

DataMapping

  • The Ready States are retrieved on the basis of the Reason Codes configured in the database for the selected equipment Type.


  • The form (1) is a dynamic form, configured using the writeback query specified in HaulWriteBack module config.
    • it is entered into the Haul Table in the database
  • 2 is a summary of how many buckets have been added by this equipment in the current shift.
    • It increments every time the operator clicks the Flic button or taps the screen


  • The TimeBar at the bottom is configured based on the reason codes available for the given equipment type.
    • Configured from Siliver Light

  • The form seen is configured using the StagingStateWriteback module config in MiTime.
