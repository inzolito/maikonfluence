Foundation App Installation

The Foundation Apps run on the Andriod 7/8/9 platform.  The Apps can be installed onto a supported Samsung Active series tablet (Active2 or Active Pro) or an Android software emulator used.

Android Emulator

  • An Android emulator such as BlueStacks can be installed onto your pc to run the apps (https://www.bluestacks.com/)

  • Download Bluestacks and install on your pc.

    You will require Local administrator access on your pc to install the software package. Contact HxGN IT for assistance.

Android App Installers

  • The Foundation App apk installers can be found on HxGN Sharepoint under the Released folder
    https://hexmet.sharepoint.com/:f:/s/MM-WW-ProductManagement/EmDROexYbilNifzEWT_BD_cBVxsmbRs1JyZifZOuy13CPg?e=d1Xhx5

Android App Installation

  • Tablet installation
    Download the apk file to your pc.  Connect the tablet via USB to your pc.  In Windows explorer, copy the apk to the Tablet device.  On the tablet, open the apk using My Files.  The App will install to the Home page.

  • Emulator installation
    Download the apk file to your pc.  Open Bluestacks on your pc.  Double click on the apk file.  The App will install to the Home page.

App Configuration

  • Open Foundation app
  • Goto the Settings icon
  • Enter the following server settings
    • Server: enterprisedemo.hexagonmining.net
    • Port: 8080
    • Name: guest
    • No password required
  • Click Save.

NOTE: The PIN for the demo environment in 1234.

  • Click on the Sync button to download system configuration to the App
