Created by Usuario desconocido (dask), last modified on jun 28, 2021

Chrome Debugger

Chrome Debugger provides the simplest way to debug a foundation App.

You can access this, by opening chrome and then navigating to chrome://inspect#devices.

By selecting inspect, chrome will open the chrome debugger which will allow you to insert break points in the source code and allow for debugging.


Alternatives

If you are using VS-Code as your code editor,  you could also use also use the Cordova-Tools extension for VS-Code, which allows to debug the code directly from the IDE.



Vysor

Vysor is not really a debugging tool, but it allows you to control the tablet directly from the desktop. It becomes quite useful, when you need to copy/paste some dummy data for testing and debugging in to the apps.
