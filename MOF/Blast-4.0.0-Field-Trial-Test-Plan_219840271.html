Fresh installation on tablet

NOTE: Reset and uninstall any already existing versions of Blast on the tablet before performing this installation!


  1. Obtain the latest Blast 4.0.0 APK
  2. Move the APK to the tablet (via a USB connection)
  3. Navigate to the APK using the tablet's file explorer
  4. Install Blast on the tablet using the APK
Blast installs successfully
Pending Testing
Installation over existing version

NOTE: It might not be necessary to obtain and/or install an older version of Blast 4.0.0 if there is already an older version of Blast 4.0.0 installed on the tablet.


  1. Obtain both an older version of Blast 4.0.0 and the latest version of Blast 4.0.0
  2. Move both APKs to the tablet (via a USB connection)
  3. Navigate to the APKs using the tablet's file explorer
  4. Install the older version of Blast 4.0.0 on the tablet
  5. Start Blast, perform the first-time setup and perform a sync
  6. Install the latest version of Blast 4.0.0 on the tablet
The newer version of Blast installs successfully and retains all setup and sync data from the older version of Blast
