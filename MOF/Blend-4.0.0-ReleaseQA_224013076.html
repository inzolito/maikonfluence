Installation over existing version

NOTE: It might not be necessary to obtain and/or install an older version of Blend 4.0.0 if there is already an older version of Blend 4.0.0 installed on the tablet.


  1. Obtain both an older version of Blend 4.0.0 and the latest version of Blend 4.0.0
  2. Move both APKs to the tablet (via a USB connection)
  3. Navigate to the APKs using the tablet's file explorer
  4. Install the older version of Blend 4.0.0 on the tablet
  5. Start Blend, perform the first-time setup and perform a sync
  6. Install the latest version of Blend 4.0.0 on the tablet
The newer version of Blend installs successfully and retains all setup and sync data from the older version of Blend

Usuario desconocido (plia)

Blend 4.0.0 changes the branding from MiPlan to Hexagon, making the app entirely different from the previously available version.
Because of this, they are installed as two separate apps, so a direct upgrade that retains setup and sync data is impossible.

Core Function (Loading)

The Loading screen has two modes: Source Display and Map Display.

Core Function (Loading)

Selecting a Blend from the Blend list on the Loading screen opens the Blend Overview modal.

The Blend Overview modal lets the user view the Stockpiles (Sources) and ROMs (Destinations). The Stockpiles list also displays the number of buckets the need to be made.

From here, the user can view the details about the Blend Recipes ("Detailed" button), start the Blend from the first Recipe step ("New" button) or continue the Blend from the last Recipe step for the user's current equipment ("Continue" button).

The user can also cancel the selection at any time with the "Close" button

  1. Login
  2. Perform the downloading steps as shown above
    1. This should place the user at the Loading screen
  3. Select a Blend from the list
  4. Open the Blend Recipe details ("Detailed" button)
  5. Observe the Blend Recipe details
  6. Close the Blend Recipe details
  7. Tap the "New" button
  8. Make a Load
  9. Logout
  10. Restart the app
  11. Login
  12. Select the same Blend from the list
  13. Tap the "Continue" button
  • The number of Blend Recipes is correct
  • The details of the Blend Recipes are correct
  • Selecting a Blend using "New" sets the number of buckets for all the Blend's Recipes to 0
  • Selecting a Blend using "Continue" retains all the previous buckets of the current equipment for the Blend's Recipes

Core Function (Loading)

The Loading screen lets the user edit Recipe Loading details.

This can be down by opening the Recipe box in the bottom-left corner and tapping the "Edit" button.

This will open the Recipe Loading Edit modal, which allows the user to change the Source and Destination of the Recipe Loading details, as well as specify if the Source is empty or to make any comments about the Load.

  1. Login
  2. Perform the downloading steps as shown above
    1. This should place the user at the Loading screen
  3. Select a Blend from the list
  4. Open the Recipe box and tap the "Edit" button
  5. Change the Source
  6. Toggle the "Recipe Src Emtpy" switch
  7. Change the Destination ("Dest." on the form)
  8. Add a comment
  9. Tap the "Save" button
  10. Make a Load
  11. Navigate to the Upload screen and perform an upload
  • The Source and Destination can be changed
  • The new Source and Destination are displayed correctly in the Recipe box
  • The "Recipe Src Empty" and "Comments" fields are correctly added and displayed in the database

Core Function (Loading)

The Loading screen lets the user make Loads.

This can be down by opening the Recipe box in the bottom-left corner and tapping the "Load" button or pushing a connected Flic button.

This will create a Load to be uploaded to the database and increase the total number of buckets performed by 1 for the current Recipe.

  1. Login

    1. This should place the user at the Loading screen
  3. Select a Blend from the list
  4. Open the Recipe box and tap the "Load" button
  5. Push the button on the connected Flic button
  • Tapping the screen saves a Load and adds 1 to the total number of buckets
  • Pushing the button on the Flic button saves a load and adds 1 to the total number of buckets
  • Reaching the required number of buckets for a Recipe makes it impossible to add more Loads for that Recipe
