Core Function

Logging-in before downloading any drill or QAQC shots and selecting the "Drilling" or "QAQC" activity from the home page results in a dialog box displaying telling the user to download a shot from a shot list.

Closing the prompt automatically opens the appropriate shot download list.

  1. Reset the app in the settings if it is not a fresh install of Drill
    1. Perform any other necessary steps (first-time setup, etc.)
  2. Login
  3. Select either the "Drilling" or "QAQC" activity from the home page
  4. A prompt should display stating that there are no downloaded shots
  5. Closing the prompt will open the shot download list
  • A dialog box is displayed upon the user selecting an activity for which there are no downloaded shots.
  • The appropriate shot download list opens upon closing the dialog box


Core Function

  2. Tap the "Manage Shots" button
  3. Tap the "Drill Shots" button on the "Select Shot Type" dialog box
  4. Tap a shot on the list
  5. Tap the "Download" button
  6. Login
  7. The shot that was downloaded is accessible in the left-hand sidebar
  • The "Manage Shots" button is available
  • The "Select Shot Type" dialog box appears
  • The shot list is populated with Drill Shots
  • Downloading a shot allows the shot to be accessed in the sidebar once the user logs-in

Core Function (Drilling Shots)

After accessing a drill shot from the left-hand sidebar, the user can create a new hole by tap-and-holding a position on the map.

This displays a prompt asking the user if the hole is a redrill. A new hole can be created by answering "No", which displays the New Hole form. Completing and saving the New Hole form creates a new hole.


  1. Access a drill shot from the left-hand sidebar
  2. Tap-and-hold any location on the map
  3. When the prompt for if the hole is a redrill appears, answer "No"
  4. Fill out the New Hole form
  5. Tap the "Save" button
  • The hole redrill prompt appears after tap-and-holding for long enough
  • Select "No" opens the New Hole form
  • Completing and saving the New Hole form creates a new hole
  • The new hole is visible on the map and available in the right-hand sidebar

  7. Tap the "Continue" button
  8. The drilling interface appears
  • The hole redrill prompt appears after tap-and-holding for long enough
  • Select "Yes" opens the prompts the user for what hole is being redrilled
  • Continuing from that point prompts the user for the name of the redrill hole
  • Continuing from there saves and creates the redrill hole and opens the drilling interface
  • The new hole is visible on the map and available in the right-hand sidebar

Core Function (Drilling Shots)

Select a drill shot hole to open the drilling interface.

Tapping on the "Flushing", "Presplit" or "Production" buttons will begin the drilling process, starting a timer in the top-right of the screen and allowing for details about the hole being drilled to be entered.

Tapping on the "End of Hole" button will end the drilling process and save the details that were entered for the hole.

  1. Select a drill shot hole
  2. Tap the "Flushing" button
  3. The drilling process begins
  4. Enter the required hole details
  5. Tap "End of Hole"
  6. The drilling process ends and the hole details are saved
  7. Repeat steps 1-6 for the "Presplit" and "Production" buttons
  • The drilling process begins when either the "Flushing", "Presplit" or "Production" buttons are tapped
  • Starting the drilling process starts the timer and allows the hole details to be entered
  • Ending the drilling process stops the timer, saves the hole details, and returns the user to the map


Core Function (Drilling Shots)

Select a drill shot hole to open the drilling interface.

If the hole needs to be redrilled, the user can commence this process by tapping the "Redrill" button. This will create a new hole (which the drilling interface automatically switches to) and allows the drilling process to start again. The user is prompted and required to enter a name for the new hole before its creation.

  1. Select a drill shot hole
  2. Tap the "Redrill" button
  3. The user is prompted to enter a name for the new hole
  4. Tap the "Continue" button
  • A prompt appears asking the user for a hole name for the new hole
  • Tapping "Continue" creates a new hole and automatically switches the drilling interface to it


Core Function (Drilling Shots)

Select a drill shot hole to open the drilling interface.

If the hole is already drilled but the details about it are wrong, the user can edit the details by tapping the "Edit" button, allowing the user to edit the details so they are correct.

Tapping the "Save" button (which replaces the list of buttons when editing) will allow the user to save new details for the hole.

  1. Select a drill shot hole
  2. Tap the "Edit" button
  3. Change some details about the hole
  4. Tap the "Save" button
  5. The new hole details are saved
  • Hole details can be edited after drilling the hole by tapping the "Edit" button
  • Edited hole details can be saved by tapping the "Save" button, which also returns the user to the map


Core Function (Drilling Shots)

Select a drill shot hole to open the drilling interface.

Tapping the "Intervals" button will change the drilling interface to the intervals interface. This interface is used to note what was found at the various intervals of the hole that was drilled, such as if there was broken ground or coal.

  1. Select a drill shot hole
  2. Tap the "Intervals" button
  3. Set some things found at different intervals of the drilling process
  • Intervals can be set for the various types of things that could be found in the ground while drilling

Core Function (Drilling & QAQC Reports)

The main pages for both the drilling report and QAQC reports contain the Performance Summary and Consumable Use Summary.

The Performance Summary allows the user to view number of hours spent ready, down, on standby or delayed, and the total number of hours of the two combined. The columns listed here only display when there is appropriate data for them. In other words, they'll only display if the user selects them for a time when Drilling or QAQC'ing.

The Consumable Use Summary has four summary boxes: Drill Bit, Hammer, Fuel, and Water. These summary boxes display a number relating to a total of a certain column from the appropriate reports. These numbers are:
- total meters used for all drill bits/hammers; and
-Â total quantity consumed for fuel/water.
These reports can be accessed by tapping the appropriate summary box for a consumable summary boxes.

  1. Access the drilling report from the left-hand sidebar
  2. Observe the Performance and Consumable Use Summaries
    1. If any summaries are empty or lacking, create some data in the downloaded drill shots
  3. Observe the total values (Performance Summary right-most column, Consumable Use Summary boxes)
  • Performance Summary columns only appear when there is data relating to a column present in that session
  • All total values are correct or within acceptable limits (due to the potential for rounding issues)
