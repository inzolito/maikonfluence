Product: MineOperate Foundation : Drill App Workflow

Created by LEADER Alan, last modified on feb 10, 2020

This App follows the same Login, settings and equipment hours functionality as the other apps.

There is a minor change to the login screen in which you can download the maps for both the QAQC and Drilling.

After logging in you will be directed to the home screen

This menu has the the app generic pre-start and equipment hours functionality aswell as the two main workflows Drilling and QAQC.

Drilling

When selecting drilling from the home menu if there are no Drill shots downloaded you will be greeted with a modal telling you to download some drill shots

This will take you to the download screen for downloading drill shots.

Select the shots that you would like to use this session and tap the download button at the bottom

After the shots are downloaded you can select a specific drill shot from the left hand pull out menu.

This menu can be opened tapping the menu button in the top left hand corner

Selecting a drill shot will navigate you to the drilling screen with that will display a map of the drill shot selected.

Existing Drill Plan

To drill from an existing drill plan find the hole on the map and tap it or find it in the right hand pull out menu.

This will take you to the drill screen

New Drill Hole

To drill a new hole that is not part of an existing drill plan tap and hold on the map at the location you would like to create the hole.
This will popup a modal asking if its a new hole or a re-drill. Tap no to proceed with drilling a new hole.

You will now be prompted to enter information about the new drill hole.

After entering the information it will take you to the normal drill view


In the drilling view you will need to select the workflow you want to drill. These are configured by the server and may be for example 'Standard' or 'Production'.
In the following example you will see the 'Standard' Workflow

Tapping one of these options will start the timer in the top right which measures how much time is spent drilling on a hole.

At the same time enabling the form on the right to enter information about the hole being drilled.

You can pause and resume this process by hitting the Workflow button you chose again that displays the pause symbol next to it.

When the hole is finished you will need to finish filling out the form on the left.
After the information is filled out you may now tap the end of hole button.

Intervals

The intervals screen can be accessed by tapping the Intervals button on the drill screen.

Here in the middle section of the screen you will see the depth of the whole represented in half meter intervals. ( This is a scroll able list )

Tapping the plus button at the end will extend the intervals based on config coming from the server.

On the right hand side you will see buttons for each of the different properties a hole can be.

When selecting one of these options you will now be able to press one of the interval segments, then the same button again or another to create a one section property or a range between the two selected segments.
In the following image you will see that we have selected Broken Ground ( symbolized by a grey border and bold font ) then tapped 0.5 and again tapped 1.50. This has created a range between 0.50 and 1.50 of broken ground.

** Please note that leaving the screen after these changes will lock them in and you will not be able to change them again returning to the screen later **


On the left hand side you will notice a graphic that just visualizes the interval segments and their properties in a method closer to the actual application.
Where the start of the hole is at the top of the screen identified by the marking 0.00 and ending at the bottom of the screen marked by the length of the intervals.

** This seems to have been intended to have some click functionality at some point but is broken and no one seems to know the history of it **

In the following image you will see that the hole intervals that have been selected will now be visualized in the left hand graphic.


** The uploading and syncing of the interval information is also in a state of disrepair at the moment **

Report

The reporting for drilling can be accessed from the left hand pull out menu, labeled Report under the drilling sub section.

This shows an overall summary for drilling information on the maps that are downloaded.

The report is broken into three sections, Drill Summary, Performance Summary and the untitled consumables.

The drill summary lists each of the drill shots with information about them as well as an unlabeled 'total' row at the bottom.

The report can be modified to group by columns. This can be achieved by tapping on the report section.

On this screen you can drag the column headers into the sectionÂ  labeled 'Drag a column header and drop it here to group by that column'
There is no back button however so you will need to navigate back via the left hand pull out menu to return to the generic report.
In the following screenshot you will see the data has been grouped by the drill method.

You can remove the Groupings you have added by tapping the 'x' on the column.

The performance summary...

The consumable summary shows each of the consumables and how many of each have been used


When any information on a drill shot has changed, you will notice an edited symbol against the drill shot in the left hand pull out menu

This identifies that there are changes that have been made locally on the tablet but not synchronized with the server.
To synchronize with the server open the upload screen from the left hand pull out menu under the drill sub heading.

The changes will also be shown here grouped under the Drilling items pending, to upload ensure that the field has a check mark next to it.
This can be toggled by tapping the category.
Pressing the upload button in the bottom right will begin the synchronization with the server

Re-Drilling

There are two methods to create a re-drill for a hole


Same Position

If the hole needs to be re-drilled in the same position due to collapse or any other issue. Navigate into the holes drill view, there you will see the re-drill button located under the workflow buttons.

New Position

If the hole needs to be re drilled in a new location, tap and hold on the map where you would like the new hole to be. A modal will appear asking if this is a new hole or a re-drill.


The re-drill option will be greyed out if you are to far away from any existing holes. The distance for this is a config variable set by the server.

If there are holes nearby and you select the re-drill option you will be prompted with a drop down to select the specific hole that you are re-drilling. This list will be populated with holes that are with in the specified distance.


You will now be prompted to give the re-drill hole a name

After this you will continue through the normal drilling process on this new hole that has been created.


Drill Consumables

There are some modifications to the basic time functionality inside Drill.

Delay options can be configured with 'Consumables' these track how much / many resources are used during drilling.

For an example

QAQC

when selecting QAQC from either the home menu or the left hand pull out menu, if there are no QAQC drill shots downloaded you will be prompted to download some.

Please select the QAQC drill shots that are useful to you and tap the Download button

You will now be able to select one of these QAQC drill shots from the left hand pull out menu.


Selecting one will take you to the map view for that QAQC drill shot

Hole Inspection

From the QAQC map view for a drill shot you can tap a hole on the map to enter the QAQC form or select it from the right hand pull out menu

The QAQC from can be seen below once selecting a hole.

Making changes to this form will automatically flag the QAQC as updated.
this will show up under the upload view from the left hand pull out menu under the QAQC sub heading.

Here you will see the updates grouped under the QAQC items pending category. To upload the data ensure the category has been selected.
This is signified by the check on the right hand side which can be toggled by tapping on the category.
Now tap the Upload button to begin synchronizing with the server.

Report

The report can be accessed by the by the report button in the left hand pull out menu under the QAQC sub heading.

Here you will see the QAQC Summary which shows how many holes have been quality assured for each of the QAQC drill shots downloaded aswell as a row at the bottom which acts as a total.

** The Performance summary and consumables appear to be a hangover from copying this report from drilling. As well as tapping on the QAQC summary takes you to the grouping page for the drilling data **
