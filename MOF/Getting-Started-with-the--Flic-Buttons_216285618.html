Product: MineOperate Foundation : Getting Started with the Flic Buttons

Created by Usuario desconocido (dask), last modified by Usuario desconocido (plia) on feb 12, 2020


Setting up

  • Install the Flic App from Google Playstore https://play.google.com/store/apps/details?id=io.flic.app&hl=en_US.
  • You will need to login to an account to use the Flic button.
    • Email: hexagonminingdev@gmail.com
    • Password: MIPlan01





  • Click on the Phone Tab at the top of the View and then the "Add Flic to Phone" button at the bottom.


 

  • Press and hold the Flic Button (You should be able to see a red light) on the button 
    • The scanner on the App should have picked up the Flic button
    • Add a name to the Flick Button
    • The Button should now be saved to your account.



  • On Loader, click the Connect button on the main Nav
    • The newly linked button should be visible.
    • Select the button will link the flic button to Loader.
    • Select one of the ready states, complete the form
      • Click the Flic button, 
      • The page should flash for a second and the number of bucket at the bottom should increment.




Tearing Down

  • The Flic buttons are locked to the tablet they are linked to.
    • At any given time, a button can be used on only one tablet.
    • So after every test, we will need to free up the button, so it can be used on another tablet.
  • To free up the button, open the Flic App and go to the Phone View.
    • Click the android icon on your flic button
      • Click release. This releases the button from Loader.
    • Click and hold the your flic button and then select remove.
      • This will allow the button to be configured on another tablet.
