Installations

TaskStepsExpectationRQA ResourceRQA NotesRQA Status
Fresh installation on tablet

NOTE: Reset and uninstall any already existing versions of Haul on the tablet before performing this installation!


  1. Obtain the latest Haul 4.0.0 APK
  2. Move the APK to the tablet (via a USB connection)
  3. Navigate to the APK using the tablet's file explorer
  4. Install Haul on the tablet using the APK
Haul installs successfully


Installation over existing version

NOTE: It might not be necessary to obtain and/or install an older version of Haul 4.0.0 if there is already an older version of Haul 4.0.0 installed on the tablet.


  1. Obtain both an older version of Haul 4.0.0 and the latest version of Haul 4.0.0
  2. Move both APKs to the tablet (via a USB connection)
  3. Navigate to the APKs using the tablet's file explorer
  4. Install the older version of Haul 4.0.0 on the tablet
  5. Start Haul, perform the first-time setup and perform a sync
  6. Install the latest version of Haul 4.0.0 on the tablet
The newer version of Haul installs successfully and retains all setup and sync data from the older version of Haul

Core Function (Reason Codes)

Select a reason code from the left-hand sidebar.

The Activity Wheel has equipment activities and names for the activities, and the default activity is "Empty".

Changing the equipment activity increases the number of time allocation activity items on the upload screen.

Completing a full cycle of the activity wheel increases the number of Hauls found underneath the Activity Wheel in Shift Summary and in the number of Hauls in the Hauling Fields.

  1. Login
  2. Select a reason code
  3. Perform some full cycles of the Activity Wheel
  4. Observe the number of Hauls in the Shift Summary and Hauling Fields
  5. Check the upload screen for the number of time allocation activity items
  6. Upload the state changes (if auto upload hasn't triggered)
  7. Check the StagingTimeAllocations and EquipmentActivities tables
  • Default activity is "Empty"
  • Completing full cycles of the Activity Wheel increase the number of Hauls in the Shift Summary and Hauling Fields
  • Changing equipment activities increases the number of time allocation activity items on the upload screen
  • All activity changes are visible in both the StagingTimeAllocations and EquipmentActivities tables
  • All activity changes are in order in the EquipmentActivities table
