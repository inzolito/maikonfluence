Product: MineOperate Foundation : Intervals Setup

Created by Usuario desconocido (kins), last modified by GEORGE Nathan on jul 12, 2021

The Drill Intervals require validation or set up of the

Dynamic Fields - Dynamic Descriptions

  • Table Name -GeoIntervalAttributes
  • Field Names - TextInterval1 ... TextInterval10, NumericInterval1 ... NumericInterval10

Data Management Query

  • HolInt - Hole Intervals ** Use the ID in front of the colon in the Mod Config writeback!

Module Configuration

  • Use the Data Management Query ID in the writeback
  • Length is 1 for 1 meter
ModuleKeyValue
MiDrillDrillIntervalWriteback4343:writeback
MiDrillDrillIntervalLength1

Standard Configuration - Lookup Values

At least one record needs to be entered into the HolInt table with the Field Code of IntervalType. Acvite is set to Yes for the record.
