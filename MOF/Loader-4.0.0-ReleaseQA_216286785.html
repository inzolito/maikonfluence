Fresh installation on tablet

NOTE: Reset and uninstall any already existing versions of Loader on the tablet before performing this installation!


  1. Obtain the latest Loader 4.0.0 APK
  2. Move the APK to the tablet (via a USB connection)
  3. Navigate to the APK using the tablet's file explorer
  4. Install Loader on the tablet using the APK
Loader installs successfully

Installation over existing version

NOTE: It might not be necessary to obtain and/or install an older version of Loader 4.0.0 if there is already an older version of Loader 4.0.0 installed on the tablet.


  1. Obtain both an older version of Loader 4.0.0 and the latest version of Loader 4.0.0
  2. Move both APKs to the tablet (via a USB connection)
  3. Navigate to the APKs using the tablet's file explorer
  4. Install the older version of Loader 4.0.0 on the tablet
  5. Start Loader, perform the first-time setup and perform a sync
  6. Install the latest version of Loader 4.0.0 on the tablet
The newer version of Loader installs successfully and retains all setup and sync data from the older version of Loader

Core Function (Reason Codes)

All Reason Codes have a "Source" and "Destination" (Dest) field.

These two fields are required and no loads can be made without filling out these two fields.

  1. Login
  2. A prompt should open. Tap the "Ok" button on the prompt.
  3. Select a Reason Code from the left-hand sidebar
  4. Tap to add a load
  5. Fill out the "Source" and "Dest" fields
  6. Tap to add a load
  • If there are Reason Codes, a prompt will open upon logging in
  • Closing the prompt opens the left-hand sidebar
  • Tapping to add a load before filling out the "Source" and "Dest" fields prompts the user to fill out the fields first
  • Tapping to add a load after filling out the "Source" and "Dest" fields adds a load to the total number of loads for the shift

Core Function (Reason Codes)

When adding a load, the user can either tap the area of the screen that adds a load or use a connected Flic button.

Adding a load will also increase the number of loads in the Shift Summary (found in the bottom-right of the screen) and underneath all the fields in the load details (found in the bottom-left of the screen).

  1. Login
  2. Navigate to a Reason Code
  3. Fill out the "Source" and "Dest" fields
  4. Tap to add a load
  5. Push the button on the connected Flic button
  • Tapping the screen adds a load to the total number of loads for the shift
  • Pushing the button on the Flic button adds a load to the total number of loads for the shift

Usuario desconocido (plia)

  • The total number of loads found under the fields in the load details resets whenever the load details are changed
    • This is present in older versions and is likely intended to display the total number for the current load details.
    • This makes more sense when considering the other total is found in the Shift Summary area, which counts the total for the entire shift.
