HaulHaul-3.1.0-2

SUPPORTED

27 Jan 2020MineEnterprise 2020.05 (3.6.1)

Internal

External

Technical Reference Manual

Module Config

English, French

Tracks Haul positions in areas with no network connectivity by sensing and storing Bluetooth beacons. The stored data is uploaded to the MineEnterprise server when the Haul truck returns to an area with network coverage. MineEnterprise Users can see the last Location the Haul Truck passed on the Operate Site Map.

Requires Bluetooth beacons to be pre-configured in the Beacon Scan app.

Location and Operator data imported and entered in MineEnterprise as usual (no changes).

Translation TBD (To Be Developed). Placeholders for Spanish, and Indonesian contain English or are blank.
