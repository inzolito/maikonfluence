MiHaulHideLockScreenSpeedDecayKmph2

Screen Lock

Set to a number between 2 to 6

  • To address poor GPS coverage
  • To unlock the Haul app when no GPS signal is detected

The Speed Decay ratio is used to estimate the amount of time needed to stop equipment when no GPS signal is available. The speed decay value is inverse

  • Lower decay value - longer time to unlock
  • Higher decay value - shorter time to unlock

After unlocking, the Haul app remains unlocked until a GPS signal can be used to calculate the movement of the equipment.

Haul 3.0.0 / ME 3.4.0

2019 May

MiHaulLockScreenOnHideIgnoreShowRequestSeconds5Screen Lock

Add or remove 2 to 5 seconds

  • To address poor GPS coverage
  • Use to keep the screen unlocked after a full stop has been detected

The lock screen setting is the number of seconds the screen stays unlocked before sampling GPS data for movement.

Increase this number by 2-5 seconds if the Haul app is looping through locking and unlocking when the equipment is at a stop or in areas with poor GPS coverage.
