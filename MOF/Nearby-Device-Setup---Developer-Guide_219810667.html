Product: MineOperate Foundation : Nearby Device Setup - Developer Guide

Created by Usuario desconocido (dask), last modified on mar 19, 2020

Add the following configuration to the promise queue, of each app requiring the Beacons.

The NearbyDevice Service, needs a list of all devices in the database to work.


{                
message:lnglm.gT('synchronisingNearbydevicemonitorDevices'),
promiseMethod: nearbydevicemonitorAdapter.getRESTDevices              
}


Adding the initialising of the Beacons in MiApp makes the app incompatible with MineEnterprise < 3.6.
