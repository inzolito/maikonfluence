StartSessionReasonCodes

 config: StartSessionReasonCodes
     Should be of format StatusCode(ReasonCategory.Code) : ReasonCode : ActivityCode?
     config Value Activity Code is optional 
     It needs to match the A ReasonCode and or activity code for that equipment Type in the database
Dig defaults to the Ready state, if there is only one set of StatusCode:ReasonCode for a given equipmentType.

If there are multiple reason Codes for the Equipment Type, the user is prompted to choose from the ready state

There is no default Value , it needs to be provided from silverlight.
