Installation over existing version

NOTE: It might not be necessary to obtain and/or install an older version of Spot 4.0.0 if there is already an older version of Spot 4.0.0 installed on the tablet.


  1. Obtain both an older version of Spot 4.0.0 and the latest version of Spot 4.0.0
  2. Move both APKs to the tablet (via a USB connection)
  3. Navigate to the APKs using the tablet's file explorer
  4. Install the older version of Spot 4.0.0 on the tablet
  5. Start Spot, perform the first-time setup and perform a sync
  6. Install the latest version of Spot 4.0.0 on the tablet
The newer version of Spot installs successfully and retains all setup and sync data from the older version of Spot

Core Function (Spotting)

The Excavator, Source Block, Material, Destination and Truck fields can have items selected from a list modal.

These list modals should be populated only with items that are valid for the appropriate fields. These list modals should also allow the user to search for items using the search bar, select all items, deselect all items, cancel the selection, or save the selection.

  1. Login
  2. Open the Excavator field
  3. Tap the "Edit Items" button
  4. Search for a valid item in the search bar
  5. Clear the search bar
  6. Tap the "Select All" button
  7. Deselect some items
  8. Tap the "Unselect All" button
  9. Select some items
  10. Tap the "Cancel" button
  11. Tap the "Edit Items" button
  12. Select some items
  13. Tap the "Save" button
  14. Repeat steps 2-12 for the Source Block, Material, Destination and Truck fields
  • The list modals for each field are populated with the correct items
  • Only items that have the string of text (case insensitive) in the search bar are visible (all valid items should be visible if the search bar is empty)
  • Using the "Select All" button selects all items in the list modal
  • Using the "Unselect All" button deselects all items in the list modal
  • Using the "Cancel" button discards all changes in the list modal
  • Using the "Save" button confirms all changes in the list modal

Core Function (Spotting)

The Load form contains numerous other fields.

The "% Remaining" and "Time Loaded" fields are automatically calculated.

The values of "Massive Percent", "Felsic Percent" and "Ultramafic Percent" fields must total to 100. The values of these fields are used to calculated the "% Remaining" field.

The "Time Loaded" field is calculated when all other non-Load form fields (Excavator, Source Block, Material, Destination, Truck) have a selected item. It is displayed on the Load form panel button on the left.

The "Comments" and "Crew" fields can be filled with any text. The "Crew" field can also be filled using the list opened by tapping the triple-line icon on the right-hand side of the text field. The icon will be greyed-out and the list will be disabled if there are no entries in the list.

  1. Login
  2. Fill out any unfilled non-Load form fields
  3. Open the Load form
  4. Observe the "% Remaining" field
  5. Fill out the "Massive Percent", "Felsic Percent" and "Ultramafic Percent" fields
  6. Re-observe the "% Remaining" field
  7. Fill out the "Comments" and "Crew" fields
    1. If available, use the list of the "Crew" field to fill it out instead
  • "Time Loaded" is automatically calculated when all other fields have selected items
  • "% Remaining" is equal to 100 - ("Massive Percent" + "Felsic Percent" + "Ultramafic Percent")
  • "% Remaining" must equal 0 for an entry to be valid (i.e. "Massive Percent", "Felsic Percent" and "Ultramafic Percent" must total to 100)
  • "Comments" and "Crew" can be filled with any text
  • "Crew" has a list that can be opened by tapping the triple-line icon

Core Function (Spotting)

The "Add Entry" button adds a new entry on the Report using the provided details.

After successfully adding a new entry, the user will be returned to the Truck field.

If there are any missing and/or invalid details, then the attempting to use the "Add Entry" button will result in a warning being displayed in the header relating what the missing and/or invalid details are.

  1. Login
  2. Fill out all the fields
  3. Tap the "Add Entry" button
  4. Fill out all the fields except for Source Block
  5. Repeat step 4 for the Material, Destination, Truck and Load form fields
  • New entries are added on the Report
  • The user is returned to the Truck field upon a new entry being successfully added
  • If any details are missing and/or invalid, the entry will not be added
  • When the above scenario occurs, a warning is displayed outlining the details that are missing and/or invalid
