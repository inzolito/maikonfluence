Product: MineOperate Foundation : TDoc - Localization and Translations

Created by Usuario desconocido (kins), last modified by Usuario desconocido (wchan) on may 30, 2022

The localizations for the MineOperate Foundation Apps are subordinate to MiApp settings, MiTime translation files, and the native App translation files.

MiApp

  • Translation enable is set to true to be built into the apps, if the locale file exists
  • Translation visibility is set to true for the language in the apps' settings. if the locale file exists

https://bitbucket.org/miplandev/miapp/src/develop/MiApp/data/language/available-language-locale.js

MiTime

  • Must contain the locale-mitime-##-##.json for each language
  • Translations of the menus, error messages, and buttons common to all apps
  • Partial and blank placeholder files exist. Translations may need to be completed

Individual MineOperate Foundation Apps

  • Must contain the locale-mi####-##-##.json for each language
  • Translations of the menus, error messages, and buttons common to all apps
  • Partial and blank placeholder files exist. Translations may need to be completed
  • A restart is required for all translations to apply
