Fresh installation on tablet

NOTE: Reset and uninstall any already existing versions of Time on the tablet before performing this installation!


  1. Obtain the latest Time 4.0.0 APK
  2. Move the APK to the tablet (via a USB connection)
  3. Navigate to the APK using the tablet's file explorer
  4. Install Time on the tablet using the APK
Time installs successfully

Installation over existing version

NOTE: It might not be necessary to obtain and/or install an older version of Time 4.0.0 if there is already an older version of Time 4.0.0 installed on the tablet.


  1. Obtain both an older version of Time 4.0.0 and the latest version of Time 4.0.0
  2. Move both APKs to the tablet (via a USB connection)
  3. Navigate to the APKs using the tablet's file explorer
  4. Install the older version of Time 4.0.0 on the tablet
  5. Start Time, perform the first-time setup and perform a sync
  6. Install the latest version of Time 4.0.0 on the tablet
The newer version of Time installs successfully and retains all setup and sync data from the older version of Time

Core FunctionBased on the type of the selected equipment, Time returns different reason codes for selection by the user.
  1. Choose a piece of equipment and take note of its type
  2. Login
  3. A prompt will appear for asking the user for their current status (reason code)
  4. Tap the "Ok" button
  5. Take note of the available reason codes for the chosen equipment
  6. Logout and repeat steps 1-5, making sure to to choose a different equipment type during step 1
  • A prompt appears asking the user for their reason code upon logging-in and completing the prestart and equipment hours (if required)
  • Different equipment types give the user different reason codes in the left-hand sidebar

Core Function (Reason Codes)

Select a reason code from the left-hand sidebar.

The equipment state buttons can be found in the footer, and the default state is "Ready".

Changing the equipment state increases the number of time allocation status items on the upload screen.

  1. Login
  2. Tap the "Ok" button on the prompt that appears
  3. Select a reason code
  4. Make some state changes
  5. Check the upload screen for the number of time allocation status items
  6. Upload the state changes (if auto upload hasn't triggered)
  7. Check the StagingTimeAllocations and EquipmentStates tables
  • Default equipment state is "Ready"
  • Changing equipment states increases the number of time allocation status items on the upload screen
  • All state changes are visible in both the StagingTimeAllocations and EquipmentStates tables
  • All state changes are in order in the EquipmentStates table

Core Function (Reason Codes)

Select a reason code from the left-hand sidebar.

The activity wheel has equipment activities and names for the activities, and the default activity is none. The number and names of these activities change depending on the currently selected reason code.

Changing the equipment activity increases the number of time allocation activity items on the upload screen.

  1. Login
  2. Tap the "Ok" button on the prompt that appears
  3. Select a reason code
  4. Take note of the number of activities and activity names on the activity wheel
  5. Make some activity changes
  6. Check the upload screen for the number of time allocation activity items
  7. Upload the state changes (if auto upload hasn't triggered)
  8. Check the StagingTimeAllocations and EquipmentActivities tables
  • Default activity is none
  • Each reason code has a different number of activities and has different activities on the activity wheel
  • Changing equipment activities increases the number of time allocation activity items on the upload screen
  • All activity changes are visible in both the StagingTimeAllocations and EquipmentActivities tables
  • All activity changes are in order in the EquipmentActivities table
