Product: MineOperate Foundation : WorkFlow

Created by Usuario desconocido (dask) on feb 05, 2020
  • Login
  • Complete the PreStart process (If required or configured)
  • Select one of the Ready reason codes ( if more than one ) for the selected equipment type
  • Select one of the Time Allocation States 
    • Ready is production
    • The other's are non-production codes

  • Complete the form , depending on the configuration from of the Writeback Query, all fields could be mandatory
  • Click the Flick Button, or tap the screen.
  • The Summary form should have incremented
    • With each further tap or click , the count on the summary form should increment.
  • Select one of the other Time Allocation States 
    • Complete the form (if required)
    • Go back to Ready state.
  • Navigate to the upload menu


  • The Count of the Hauling Items should be the same as that on the Shift Summary.



  • The count of the Time Allocation Status should be same as the number of TimeAllocation Statuses selected.


  • Upload the data to the server.
    • Ensure the data is uploaded.
  • The records should be in the Haul table and the StagingTimeAllocations in the database,
