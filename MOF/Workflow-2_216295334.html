Product: MineOperate Foundation : Workflow 2

Created by LEADER Alan, last modified on mar 13, 2020

The Spot app uses the standard settings and start session funcitonality from MiApp.

It Does not however use MiTime, prestart or vehicle hours.

After login you will be greeted with this screen.


There is no home screen as this is the main use of the App, the only other is review which we will touch on later.

There are a configurable amount of fields on the left hand side, In this case which is the default the excavator and Load screens function slightly differently.

All the other fields are children of the excavator field. So when changing the excavator you will see them all change to empty. The options that have been downloaded for each category will persist per excavator too.

So lets begin by downloading some excavators to select. Do this by tapping the "Edit Items" button, this will bring up the following modal

This is a list of possible excavators on the server, select the ones that potentially apply to you then hit save. You will then see the ones you have selected for download appear as below.

You can now select one of these excavators to begin filling out the rest of the form

each of the next categories functions the same except will not reset the selection of other categories when changed.

Finally when selecting the Truck you will see that it sets the load time under the load category.

It is assumed that the truck has been loaded at the time that you selected the truck.

Now the Load Form.

Here you will need to fill out extra information specific to the load just completed. This information is from a dynamic query and can be configured.

Something special to this app is formulas for fields. In theis case the Massive Percent, Felsic Percent and Ultramafic Percent need to add up to 100%. in the field below them you will see how much is remaining until this goal is reached.

without hitting this target and trying to add the entry you will see the error as below.

These fields are configured as a json array of fields in the module config ValidationFields

when the form is complete you can add the entry.


After adding the entry you will be returned to the truck selection. This is because it is assumed the geologist will be in the same excavator continuing in the same spot with only the truck and the load changing.

From the Primary pull out menu you can select the report tab. This contains a table of all the loads that have been completed for this tablet.

Here you can group the columns to see the data in a way that is useful to your purpose.


There is also the edit button in which you can modify the information for a load that has occured in the past. This will pop out a modal that is the same form from the Load category in the spotting workflow.
