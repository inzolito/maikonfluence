Product: MineOperate Foundation : Workflows

Created by Usuario desconocido (dask), last modified on feb 09, 2020

Login Workflow

  • Similar to the other Apps,


  • The login screen has the additional functionality to allow users to download blast patterns
  • If no blast patterns are downloaded, the App will prompt you to download a pattern



Patterns can be downloaded from the Download View, accessed from the side nav


Hole Loading

  • Once a pattern has been download, use the secondary menu to select a blast Hole.



  • Use the deck (1) to select different parts of the blast hole, and a complete the form (2) as required.
  • Each part of the deck is linked to a different type of form. The one above is the non-Explosive form
  • The Explosive forms allow users to add CupWeights.
  • Once a cup is added to a BlastHole, a red dot appears next to the Hole on the map
  • Cup Weights can be edited further using the Cup Weights page from the main navigation menu

  • New measurements for a cup weight can be added only to the most recently added cup weight.


Blast Summary

  • Similar workflow to Hole Loading.
  • Download a blast pattern
  • Select the Location will load up the Blast Summary form


  • This is populated using the Blast Summary writeback query defined in Silverlight.
  • The Blast summary provides a summary of all blast's that have been performed on a given pattern.
