This procedure installs the latest .mender update and application structure for the Display-X using a single-core environment.




Copy the full USB key structure from the server to the Display-X in the directory:
/var/lib/masep-manager/installfiles





üí° Note: You can copy the USB structure to the server first, then transfer it to the Display-X.

sudo scp -r¬†jigsaw@192.168.14.21://opt/Jigsaw/Releases/common-x86/apps-allarch /var/lib/masep-manager/installfiles/

sudo scp -r jigsaw@192.168.14.21://opt/Jigsaw/Releases/common-x86/hmcmd_config /var/lib/masep-manager/installfiles/

sudo scp jigsaw@192.168.14.21://opt/Jigsaw/Releases/common-x86/masep-display-displayx-3.5.1.qa-r3b28e81f-b4.mender /var/lib/masep-manager/installfiles/






If these folders below do not exist on your USB key, please create them. (They can be empty.)



sudo mkdir -p /var/lib/masep-manager/installfiles/apps-custom
sudo mkdir -p /var/lib/masep-manager/installfiles/hmcmd_custom





Execute Installation

Run the installer script on the Display-X:

sudo python3 /usr/src/masep-manager/hmcmd-installer.py -s DT620 -l -v

  • -s = Executes the installation using hmcmd.conf for the specified equipment name.

  • -l = Starts the installation using the files located in /var/lib/masep-manager/installfiles.

  • -v = Runs the installation in verbose mode.





After entering the command above, you will notice that the Display screen begins reinstalling the entire application. Wait until the installation is fully completed.


If any error occurs, use the following command to clean all contents and avoid leftover data, then repeat the steps:


sudo masep-clean -y
