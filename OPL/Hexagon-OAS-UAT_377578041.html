1. Introduction

This document aims to validate the implementation of the Operator Alert System (OAS) in accordance with the requirements defined in this UAT.

2. UAT Scope

  • Installation and configuration of OAS-HV hardware
  • Validation of fatigue and distraction events
  • Communication with local/cloud server
  • Validation of OpWeb/OASWeb reports
  • Acceptance Criteria

Hardware:

  • Devices correctly installed (cameras, speakers, vibrators)
  • Correct camera positioning
  • Operation of indicator LEDs

Software:

  • OpenWRT installed correctly
  • Software in the correct version (ex. 7.X)
  • Patches and updates applied

Events:

  • Microsleep correctly detected (eye closure > 50%, >1.5s)
  • Distraction correctly identified (angle <60°, >75% in 5s)

Reports and Interface:

  • Daily, weekly, and monthly reports correctly generated
  • Data correctly displayed
