System Installation - OP Pro Server 4.5

Overview

OP Pro 4.5 will be tested to support integration between OP Pro and OAS to use a single core hardware.

This document highlighting procedure when updating OP Pro server only from version 4.5.


Note: Although the file names are labeled with version 5.0, please note that the server will be configured with version 4.5. Therefore, all validations, compatibility checks, and testing should be based on version 4.5, disregarding any nominal references to version 5.0 in the file names.
