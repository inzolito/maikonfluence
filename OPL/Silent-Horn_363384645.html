OP PRO LATAM : Silent Horn

Created by JUNIOR Nelson, last modified on nov 26, 2024

In mining operations, shovel operators commonly sound the horn to signal the truck driver that loading is complete and they should start hauling to the assigned dump. However, in mines that operate in close vicinity to noise sensitive areas, such as residential areas, the noise from the horn can cause a nuisance. The OP PRO system can address these issues by notifying the truck driver that loading is complete through features like the Silent Horn.

Silent Horn Functionality: This feature allows a system message to be sent from the Shovel operator to the Truck operator to notify them the loading process is complete. This eliminates the need to sound a horn.  The ability to send a notification to the Truck has been integrated with the Shovel kick out switch, negating the need for the Shovel operator to interact with the UI.

Upon reaching the shovel's radius and completing the spotting point, the shovel changes its activity from "Waiting" to "Loading," initiating what is called the "Load First Bucket." The truck then detects the first dipper or manual cycle, after which its activity changes to "Loading." This process repeats until the shovel operator determines that the truck is fully loaded..

Truck operator screen

Shovel operator screen


Once the truck is fully loaded, the shovel operator triggers the kick-out footswitch, sending a message to the truck operator stating: "Loading complete." Upon viewing the received message, the truck begins moving and updates its activity to Hauling.

Truck operator screen

Shovel operator screen

After completing the loading cycle, the Silent Horn function eliminates the need to sound the horn, as requested by the customer.
