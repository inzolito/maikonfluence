Product: MineOperate OP Pro : How Will The Change From Non-Production Activities To Production Activities Occur?

Created by NIELSEN John, last modified on feb 23, 2021
QuestionHow will the change from Non-production activities to Production activities occur?
AnswerThe change from Production activities to Non-production activities will be updated using activity detection on the panel. e.g. if tipping, then assigned to tie down for shift change JAMS will created a moving empty activity. Panel will update dynamically to non-production activity wheel. For Non-Production to Production activities operater can select the Production activity (tipping or hauling) if not auto detected.
