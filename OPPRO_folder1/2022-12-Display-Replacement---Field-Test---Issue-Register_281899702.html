13 ene 2023 

3.0-003 Web panel does not have brightness control

V1000 does not have hard buttons for brightness control

Brightness control will not be added to 3.0-003 web panel as this is regret effort

Sites will be required to deploy/upgrade to OP Pro 4.0 for brightness control in Shell App x64

Brightness control exists in Shell App.  

OP Pro 4.0 Shell App x64 will support V1x00 Displays

Workaround in place at AMM to set brightness via Chron job on a daily basis for 3.0-003 web panel

MASEP Audio API to be updated

Buzzer volume control to be supported in Shell App x64
