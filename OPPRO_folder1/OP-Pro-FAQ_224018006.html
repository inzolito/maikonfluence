How Will The Change From Non-Production Activities To Production Activities Occur?How will the change from Non-production activities to Production activities occur?The change from Production activities to Non-production activities will be updated using activity detection on the panel. e.g. if tipping, then assigned to tie down for shift change JAMS will created a moving empty activity. Panel will update dynamically to non-production activity wheel. For Non-Production to Production activities operater can select the Production activity (tipping or hauling) if not auto detected.

ANSWERED
